# Project Context

## Overview
This project is a REST API server built with Python.

## Architecture
- **Web Framework**: FastAPI
- **Database**: PostgreSQL
- **ORM**: SQLAlchemy
- **Documentation**: Redoc

## Objectives
- Build a robust REST API.
- Maintain clear and comprehensive API documentation.
